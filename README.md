# Fundamentals II starters

(Unofficial) Starter files for [Fundamentals II - Introduction to Class-based Program Design](https://course.ccs.neu.edu/cs2510sp22/) course.

The course also references exercises in ["How to Design Classes" book by Felleisen et el.](https://felleisen.org/matthias/htdc.html)

## Contributing

**N.B.** Please note that the solutions in this repository are intended as study aids for the course exercises. *My code can (and most probably does) contain mistakes.* As such, I highly welcome and encourage any pull requests that fix errors or improve the solutions. Your contributions will help make this repository more accurate and useful for everyone!
